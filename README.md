# calculator-react
A basic react calculator built in react js 

### Installing

* Install npm: https://www.npmjs.com/get-npm
* Install nodejs: https://nodejs.org/en/download/

### Executing program

* clone the repo
* install node_modules
* launch the service
```
npm install
npm run start
```
